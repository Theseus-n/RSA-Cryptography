Usage Guide

RSA Encryption:

  • Encrypt: Input plain text (example: HELLO)
  • Decrypt: Input ciphertext numbers (example: 186 114 200 200 203)

Caesar Cipher:

  • Encrypt/Decrypt: Input text
  • Shift: Number between 1-25
  • Example: Shift 3, "ABC" → "DEF"

SHA-256 Hashing:

  • Hash operation only (no decryption)
  • Example: "hello" → "2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824"
